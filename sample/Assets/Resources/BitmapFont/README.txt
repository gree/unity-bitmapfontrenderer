How to create font.bytes, font_texture.png.

1. Execute fontconv.rb.
  $ macruby ../../../../tools/fontconv.rb [fontFile] [fontName] [Size] "" [TextureWidth] [TextureHeight] font

2. Import font_texture.png.
Texture Inspector
  Texture Type: Advanced
  Generate Mip Maps: NO
  Max Size: 2048
  Format: Alpha 8

--
How to generate the sample data.
  $ macruby ../../../../tools/fontconv.rb MTLmr3m.ttf MotoyaLMaru 17 "" 2048 2048 MTLmr3m
  $ macruby ../../../../tools/fontconv.rb Tuffy_Bold_Italic.ttf Tuffy 50 "" 2048 2048 Tuffy
